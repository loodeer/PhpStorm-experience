版本控制

在可视化界面使用 Git ，PhpStorm 是我见过最好用的 Git 管理软件 😯。

解决冲突：

⌘K` Commit Changes 提交修改

⌘D Show Diff 查看差异

⌘⇧K Push Commits 推送修改

Changelist 也是个非常好用的功能，特此提一下，使用很简单不多说。
